# mule

Headless API built with Express + Prisma + SQLite.

## Getting started

```bash
npm install
npx prisma migrate dev --name init
npm run dev
```

Server runs on `http://localhost:3000`.

## Models
- User

## API Endpoints
- `GET    /users`
- `GET    /users/:id`
- `POST   /users`
- `PUT    /users/:id`
- `DELETE /users/:id`
